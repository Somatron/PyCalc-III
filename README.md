PyCalc III KU 127
---------------
Engine includes:
- Vector Operations
  - Dot Product
  - Cross Product
- Equations of lines and planes
  - Parametric Equations
  - Symmetric Equations
- Vectors Verification and Geometry Space
  - Check if vectors are Orthogonal
  - Check if vectors are Parallel
  - Calculate Vector Parallelogram
- Partial Derivatives (2 & 3 variables)
- Double Integrals (2 & 3 variables, treats Z as constant)

--------------------------------------
USE CASE
--------------------------------------
I didn't add too many comments as I usually do whenever I program, this was made in 1 day and just wanted to get general programming practice

To run it just run main and input whatever vector coordinates needed

Use it if you really dont want to do Cross Product or Vector calculations in your head

--------------------------------------
